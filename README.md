⚡ FX Multi-Timeframe Matrix：操作マニュアル
このツールは、FX市場における通貨ごとの相対的な強さを可視化し、効率的な通貨ペア選びをサポートします。

🛠 使い方
時間足の選択: 左側のドロップダウンメニューから、分析したい時間足（1分足〜日足）を選択します。

判定期間の設定: スライダーを動かし、過去何本分のローソク足データを比較対象にするか決定します（5〜50本）。

強弱ランキングの確認: 画面中央に表示されるカラーバーを確認します。左側（赤）に近いほど「強く」、右側（灰）に近いほど「弱い」通貨です。

推奨トレードの確認: ランキングに基づき、最も勢いのある通貨と最も売られている通貨を組み合わせた「推奨ペア」と「売買方向」が自動表示されます。

チャートで詳細分析: 「チャートを見る」ボタンをクリックすると、Yahoo Financeの該当ページへ移動し、詳細な値動きを確認できます。

データの更新: 最新の状況を反映したい場合は、画面下の「データ更新」ボタンを押してください。
